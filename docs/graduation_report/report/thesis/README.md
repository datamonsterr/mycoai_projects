# MycoAI Graduation Thesis — LaTeX Template

OpenPrism-compatible `report`-class template. Vietnamese encoding (utf8), BibLaTeX references, glossary support.

## Quick Compile

```bash
pdflatex main && bibtex main && pdflatex main && pdflatex main
```

## Requirements

- `texlive-full` (pdflatex, bibtex, glossaries)
- Or `tectonic` for lighter builds
