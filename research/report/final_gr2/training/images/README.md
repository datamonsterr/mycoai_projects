## Image Sources

### Training Curves
- **Source**: `/weights/*_training_history.png`
- **Generated by**: `train_models.py` during fine-tuning
- **Content**: Training/validation accuracy and loss plots

### Model Comparison
- **Source**: `report/training/all_result.png`
- **Generated by**: Manual analysis of evaluation results
- **Content**: Bar chart comparing all feature extractors

### t-SNE Visualizations
- **Source**: `report/data_visualization/tsne_finetuned_*.png`
- **Generated by**: Dimensionality reduction analysis script
- **Content**: 2D projections of 2048-dimensional feature space

### Confusion Matrices
- **Source**: `results/run_20260128_220536_k7/*/confusion_matrix.png`
- **Generated by**: `evaluate_species.py` during evaluation
- **Content**: Per-species classification performance

### Prediction Visualizations
- **Source**: `results/run_20260128_220536_k7/efficientnetb1_finetuned_weighted_all/visualizations/`
- **Generated by**: `evaluate_species.py` with `--visualize` flag
- **Content**: Query images + k=7 retrieved neighbors with similarity scores

## Usage

All images are referenced in `FINETUNING_REPORT.md` using relative paths:

```markdown
![Description](images/filename.png)
```

## Reproduction

To regenerate these visualizations:

1. **Training curves**:
   ```bash
   uv run python -m colab.train_models
   ```

2. **Upload fine-tuned features**:
   ```bash
   uv run python -m src.scripts.upload_finetuned_features
   ```

3. **Evaluate and visualize**:
   ```bash
   uv run python -m src.main evaluate-species \
     --feature-extractor efficientnetb1_finetuned \
     --strategy weighted \
     --environment all \
     --visualize
   ```

4. **Generate t-SNE plots**:
   ```bash
   # Run your dimensionality reduction analysis script
   uv run python -m src.scripts.visualize_tsne
   ```
